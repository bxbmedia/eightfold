Workflow

Your Path to Mastery

Follow these steps to create your portfolio and level up your Git skills.

1

Clone the Repository

Open your terminal and clone the project to your local machine. This downloads a full copy of the repository.

git clone https://github.com/bxbmedia/eightfold.git

Then navigate into the project folder.

cd eightfold
2

Pull the Latest Changes

Before starting any work, make sure you have the most up-to-date version of the project. Run this anytime you want to sync with what others have pushed.

git pull origin main
3

Create Your Branch

Create a new branch for your work. This keeps your changes separate from everyone else's until you're ready to merge.

git checkout -b add-[your-name]-portfolio
4

Copy the Template

In your IDE (VS Code, Cursor, etc.), find the portfolios/template folder in the file explorer. Right-click it, copy it, and paste it inside the portfolios/ directory. Rename the copied folder to your first name (e.g. portfolios/jordan).

5

Build Your Portfolio

This is your page ‚Äî you have full creative control. Open portfolios/[your-name]/index.html and make it your own. Fill in your project details, add your tech stack, write your reflections, and customize the styling however you like.

6

Update the Homepage

Open the root index.html file and find your card in the Team section. Update your tagline and change the link to point to your portfolio folder.

7

Commit & Push

Stage all your changes, write a clear commit message describing what you did, and push your branch to GitHub.

git add . && git commit -m "Add: [Name] portfolio" && git push
8

Open a Pull Request

Go to the repository on GitHub and open a Pull Request to merge your branch into main. Request a code review from a teammate.

gh pr create --title "Add: [Name] portfolio"
